# :memo: Commit Convention

```
  - add : 파일 추가
  - remove : 기능, 파일 삭제
  - feat : 기능 추가 , 업데이트
  - update : 기능, 디자인, 파일 업데이트
  - fix : 버그 수정
  - big fix : 커다란 변경의 경우 !
  - comment : 필요한 주석 추가 및 변경   
  - test : 테스트 추가, 테스트 리팩토링(프로덕션 코드 변경 X) 
```
